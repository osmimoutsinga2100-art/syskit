# SysKit - Outil d'Administration Linux

Projet de programmation Shell Bash
DUT-1 CI - CUK de Koulamoutou

## Description

SysKit est un outil d'administration Linux écrit en Bash.
Il permet d'automatiser les tâches courantes d'un système Linux.

## Structure du projet

syskit/
├── README.md
├── syskit.sh
├── modules/
│   ├── inventory.sh
│   ├── backup.sh
│   ├── cleanup.sh
│   └── report.sh
├── logs/
├── backup/
├── reports/
└── docs/
    └── rapport_final.pdf

## Utilisation

# Informations système
./syskit.sh inventory

# Sauvegarder un dossier
./syskit.sh backup /chemin/vers/dossier

# Nettoyer un répertoire
./syskit.sh cleanup /chemin/vers/repertoire

# Générer un rapport
./syskit.sh report
